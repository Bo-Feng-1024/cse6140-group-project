Please also note the autograder requirements in the current handout. The autograder checks are:

BnB: exact correctness on the test instances

Approx: validity of the returned vertex cover on all released test, small, and large instances

LS1: reaching the value in the corresponding .out file on all test and small instances, with a 2-second cutoff per run

LS2: reaching the value in the corresponding .out file on all test and small instances, with a 2-second cutoff per run

Please read the current handout carefully before implementing or running experiments.
